Hardware

I work on a MacBook Pro. The computer has 2.2 GHz Intel Core i7 with 8 GB of RAM. I use a second generation iPad while working to browse documentation, to log into remote servers, and to check email.

Coding

I write code in Emacs inside of iTerm2. My shell of choice is zsh, which I configure with the help of oh-my-zsh.

I use AppleScript Editor to write AppleScripts.

I use Xcode to manage my iOS devices and to design iOS interfaces.

Languages

I write Ruby, lots and lots of Ruby. My default Ruby is JRuby. I use MRI version 1.9.3. I manage my Ruby versions with rbenv.

I use MacRuby to mock up Mac applications and RubyMotion to write iOS applications.

I love functional languages like LISP. My brain fits the functional paradigm. My preferred LISP is Clojure.

Halfway between Ruby and LISP is JavaScript. I prefer CoffeeScript.

Services

I use GitHub as my code repository. I use GitHub's post-receive hooks to kick off continuous integration jobs at CloudBees and TravisCI.

I deploy applications to Heroku, Engine Yard, and RedHat OpenShift.

Writing

I write notes in Markdown in Emacs on my Mac and in Elements on my iPad. I save my notes to Dropbox.

Utilities

I use Safari as my web browser, with the occasional use of Chrome.

I use Twitterrific to read and manage my Twitter feeds.

I use Colloquy as my IRC client on the Mac and on my iPad.

NetNewsWire continues to serve as my news reader.

I use Alfred to launch scripts, search documentation, navigate the file system, and sundry other things.
